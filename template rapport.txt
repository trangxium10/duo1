\section{Introduction}

Dans ce projet, nous avons réalisé un mini site e-commerce permettant de gérer des produits en ligne.

L’objectif est de développer un site web dynamique en utilisant :
\begin{itemize}
    \item PHP
    \item MySQL
    \item HTML / CSS / JavaScript
\end{itemize}

%--------------------------------------------------
\section{Présentation du site}

Le site permet d’afficher une liste de produits et de les gérer.

Un utilisateur peut consulter les produits, tandis qu’un administrateur peut :
\begin{itemize}
    \item Ajouter un produit
    \item Modifier un produit
    \item Supprimer un produit
\end{itemize}

%--------------------------------------------------
\section{Fonctionnalités}

Les principales fonctionnalités du site sont :

\begin{itemize}
    \item Affichage des produits (SELECT)
    \item Ajout d’un produit (INSERT)
    \item Modification d’un produit (UPDATE)
    \item Suppression d’un produit (DELETE)
\end{itemize}

\section{Base de données}

Nous avons créé une base de données appelée \textbf{ecommerce}.

\subsection{Table produits}

\begin{tabular}{|c|c|c|}
\hline
Champ & Type & Description \\
\hline
id & INT & Identifiant \\
nom & VARCHAR & Nom du produit \\
prix & FLOAT & Prix \\
description & TEXT & Description \\
\hline
\end{tabular}

\subsection{Requête SQL}

\begin{verbatim}
CREATE TABLE produits (
id INT AUTO_INCREMENT PRIMARY KEY,
nom VARCHAR(100),
prix FLOAT,
description TEXT
);
\end{verbatim}
